FacultyInformationScraper
=========================

Parses the faculty information of all of VIT University's professors and generates a JSON dump.

# Requirements

* args==0.1.0
* BeautifulSoup==3.2.1
* clint==0.4.1
* mechanize==0.2.5
* Pillow==2.9.0
* wheel==0.24.0
